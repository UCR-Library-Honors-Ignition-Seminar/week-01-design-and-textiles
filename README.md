## Session 2 (Day 2): Vector Mechanics & Textile Construction

### Inkscape & Vector Design

A **vector file** (SVG) is made of mathematical paths — it scales to any size without blurring. Laser cutters and Cricut machines use SVG files.

**In-class activity:** Open Inkscape, load the provided "Legend of Zelda Shield" starter file, practice nodes and paths, export as SVG.

- Inkscape download: [inkscape.org](https://inkscape.org) (free)
- Inkscape tutorials: [inkscape.org/learn/tutorials](https://inkscape.org/learn/tutorials/)

### Hand-Sewing Basics

| Stitch | Use |
|--------|-----|
| **Ladder stitch** | Closing openings invisibly |
| **Running stitch** | Quick joins |
| **Backstitch** | Strong permanent seams |
| **Button sewing** | Reattaching buttons |

**Activity:** Construct a rice-filled hand warmer using patterned cloth.
